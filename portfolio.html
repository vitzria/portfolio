    Calculadora IMC

    Projeto feito com HTML, CSS e JavaScript de uma calculadora de Índice de Massa Corporal. O usuário dá a entrada com o peso (em Kg) e a altura (em M), o Sistema retorna o valor do IMC e como ele se classifica (baixo peso, peso adequado, sobrepeso, obesidade grau 1, obesidade grau 2 ou obesidade extrema). A aplicação pode ser útil para profissionais da saúde, educadores físicos ou pessoas que querem saber se estão no peso ideal. Repositório do projeto no GitHub

    Lista de afazeres

    Projeto de aplicação feito com HTML, CSS e JavaScript de uma "to-do-list", uma lista de afazeres. O usuário pode listar todas as tarefas que ele precisa fazer. Como funcionalidades é possível finalizar uma tarefa (deletando) e editar uma tarefa. É uma aplicação básica que pode ser usada por qualquer pessoa que queira organizar os seus afazeres.
